# Nebula
Nebula is a ECS 2D Game Engine written in C++ with a Lua API. Nebula is heavily inspired by LOVE2D, flecs, bevy and Lotus.

Please check the [documentation](https://github.com/maumafra/nebula/tree/main/documentation) page.
